# Number Guessing Game

## Description
A single-player terminal-based game where you guess a random number between 1 and 100.

## How to Run
1. Make sure you have Python installed
2. Run the command: `python game.py`

## Features
- Random number generation (1-100)
- Feedback for too high / too low
- Tracks number of attempts
- 10-attempt limit
- Input validation
- Play again option

## Data Structures Used
- Integers for number and attempts
- While loop for game logic
- If-elif-else for comparisons
